Iframe 跨域嵌套页面，本地模拟

Mar 29, 2017 | 技术 | Hits

什么是跨域？以下是跨域的情况。

  • 同一域名，不同端口
  • 同一域名，不同协议
  • 域名，和域名对应的ip
  • 主域相同，子域名不同
  • 同一域名，不同的二级域名
  • 不同域名

我们在开发中，尤其是写组件的时候，经常会用到iframe来嵌套第三方的组件。有时候我们也提供组件给其他业务方，这个时候不可避免会遇到跨域的问题。
虽然我们有测试服务器可以测试，但是本文提供一种本地的模拟跨域，可以用来测试自己的代码和用例

笔者使用的是mac笔记本自带的 apache 服务器，和 anywhere 静态服务器，启动之后端口不一样，实现本地跨域。

本文旨在提供一个思路，即，本地的端口不同，来模拟跨域。具体的服务器配置请自行搜索资料补脑
