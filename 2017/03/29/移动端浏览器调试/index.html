移动端浏览器调试

Mar 29, 2017 | 技术 | Hits

我们做移动端开发的时候，经常需要debug一些问题，简单总结一下移动端浏览器的调试方法

ios + safari 苹果调试

  • 常用的调试苹果手机浏览器Safari的办法
  • 打开mac的Safari和苹果手机的Safari浏览器，数据线连接
  • mac上面的浏览器点击 开发 选项，选择列表用户mac的一项
  • 选中自己手机的页面即可开始调试

chrome浏览器调试

1.mac 上安装chrome浏览器，我用的是版本 v56.0.2924.87 (64-bit)。
2.手机上安装chrome浏览器，我下载的是 v56.0.2924.87
3.一条数据线，连接手机和电脑，手机打开调试模式
4.mac上的chrome浏览器打开地址：chrome://inspect/#devices 即可看到手机chrome
浏览器访问的地址

PS：建议pc和手机的chrome版本一致，都从chrome官网上面下载。我发现手机浏览器的版本比pc的高
的话，pc会获取不到页面

UC浏览器调试

UC浏览器开发者中心

  • 据我所知，目前这种方法是只能在windows系统下实现
  • window下载uc开发者版本，下载豌豆荚
  • 手机连接开发者模式，chrome浏览器: chrome://inspect/#devices 即可看到手机页面
  • 参考文档

最后一弹

  • Weinre
  • 手机和电脑处入同一ip网段下
  • 在Rules 的选项里面 Default 里面加上你要调试的 url+ weinre://test
