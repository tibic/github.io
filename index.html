Mac下的svn服务器，svn命令

2017-03-29

我们习惯使用git的命令行，其实svn也有命令行，需要在mac本地搭建svn服务器。

MAC下的SVN命令行

svn co path //将文件checkout到本地目录
svn copy trunk path //创建新分支
svn revert path //恢复原始为改变的工作副本文件，恢复大部分的本地修改。不能恢复删除的目录
svn up //更新到最新版本
svn up -r m(版本号) path ;如svn up -r 200 test.php //将版本库中的文件test.php还原到版本200
svn status //查看文件目录状态
svn delete path //删除文件
svn commit -m “描述” //提交文件添加说明
svn log path //显示文件的修改记录和版本号的变化
svn info path //查看文件详细信息
svn diff path //比较差异
svn merge -r m:n path //将两个版本的差异合并到当前文件
svn list path //显示path目录下的所有版本库的文件和目录
svn mkdir path //创建纳入版本控制的新目录
svn lock -m “说明” path //加锁
svn unlock path //解锁
svn revert -R something,当something为目录时候恢复删除目录（改动没有提交）
svn revert something,当something为单个文件时恢复删除文件（改动没有提交）
svn cleanup //解锁locked
svn revert –recursive example_folder 撤销add
